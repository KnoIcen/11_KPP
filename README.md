# Screen Recorder (LR11)

Electron app for desktop screen capture and WebM recording.

## Run

```bash
npm install
npm start
```

## Build installer (Windows)

```bash
npm run make
```

Installer output: `out/make/squirrel.windows/x64/`

## Usage

1. Click **Вибрати джерело відео** and pick a screen or window.
2. **Старт** / **Стоп** to record.
3. Save the file in the dialog after stop.
4. **Відкрити відео** to play a saved file.
